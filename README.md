# IT_otazky

Rozšířené opakovací poznámky k odborným otázkám IT státnic (okruhy 1–12).

- `main.tex` – kompletní materiál v LaTeXu, sázeno pdfLaTeXem (stejný styl jako repozitář `BSS_PRAVO`).
- Klíčové pojmy jsou v textu zvýrazněné tučně, doplněné o seznamy, příklady, kódové ukázky a přehledové tabulky.
- Pokryté okruhy: výpočetní systémy, operační systémy, souborové systémy, sítě, síťové aplikace a bezpečnost, principy programování, databáze, softwarové a informační systémy, základní bezpečnostní funkce, autentizace a řízení přístupu, analýza a řízení rizik, bezpečnostní architektura.

## Překlad
V Overleafu nastav kompilátor **pdfLaTeX** a stiskni Recompile. Lokálně:

```
latexmk -pdf main.tex
```
